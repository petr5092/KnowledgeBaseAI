# План: Реализация генератора учебных программ (Curriculum Generator)

Для создания учебных программ под разные задачи (например, "Onboarding", "Exam Prep", "Skill Upgrade") предлагается внедрить автоматизированный генератор на базе LLM. Это позволит создавать персонализированные треки обучения поверх существующего графа знаний.

## 1. Новый сервис `CurriculumBuilder`
Создать модуль `backend/app/services/curriculum/builder.py`.

**Функционал:**
*   **Input:** Цель обучения (Goal), Целевая аудитория (Audience), Доступные предметы (Subjects context).
*   **Logic:**
    1.  Запрашивает у Neo4j список доступных тем (Topics) и навыков (Skills) по выбранным предметам.
    2.  Отправляет промпт в LLM: *"Выбери из списка темы, необходимые для достижения цели '{Goal}', и выстрой их в логической последовательности"*.
    3.  Получает JSON со списком UID тем.
*   **Output:** Структура для сохранения в БД (код программы, название, список узлов).

## 2. Расширение API `admin_curriculum.py`
Добавить эндпоинт `POST /v1/admin/curriculum/generate`.

**Параметры запроса:**
*   `goal`: "Подготовка к собеседованию Python Junior"
*   `audience`: "Студенты"
*   `subject_uids`: ["SUB-PYTHON", "SUB-SQL"] (опционально, для сужения поиска)

**Результат:**
*   Создает запись в таблице `curricula`.
*   Заполняет `curriculum_nodes` выбранными темами.
*   Возвращает `curriculum_code`.

## 3. Создание дымовых (Demo) данных
Чтобы продемонстрировать работу, мы выполним скрипт/последовательность вызовов:
1.  **Ingestion:** Сгенерируем предмет "Корпоративная безопасность" (через `CorporateStrategy`).
2.  **Generation:** Создадим два курикулума на базе этого предмета:
    *   *"Инструктаж для новых сотрудников"* (Только основы).
    *   *"Сертификация офицера безопасности"* (Углубленный курс).
